Descrição: exercício em dupla ou trio com intuito de 
modificar um site simples hospedado no gitpages. 
As alterações devem seguir um dos temas descritos 
mais a baixo.


Antes de inciar deve-se definir quem vair ser o dono do 
repositório ele deve fazer o seguinte:
- Fazer fork deste repositório 
https://github.com/aluno-jvc/template-html-css
- Ir em Settings/Pages - Build and deployment e dar save na main
- Ir em collaborators e convidar os integrantes 


Tarefas: Esse exercício é separado em tarefas, 
cada integrante do grupo deverá pegar uma tarefa 
por vez para resolver. Para cada tarefa deve
ser criada uma branch e ser seguida a dinâmica 
aprendida (commits, pull requets, merge)


A) Alterar o nome da página dentro da tag <title>
B) Alterar o título dentro da tag <h1>	
C) Alterar o parágrafo dentro da tag <p>
D) Alterar o link dentro da tag <a>

E) Alterar a cor de fundo no rgb do background-color do body
F) Alterar a cor do título no rgb do h1
G) Alterar a cor do título no rgb do p

I) Alterar o titulo do README
J) Alterar o texto do README
K) Adicionar o link do github dos integrantes

Dicas: 
1- Tentem não editar o mesmo arquivo ao mesmo tempo, 
então separem bem as tarefas entre os integrantes. 
Caso isso ocorra aparecerá um conflito na hora do merge,
pare resolver, chamem o joão.
2- Cuidado ao fazer o pull request, 
na base pode aparecer outro repositório.
